# Angel2500

Этот проект представляет собой основу для **AI-системы**, которая использует **OpenAI**, **Binance**, **Bybit** и другие источники данных для работы в веб-приложении.

## Требования
- Flask
- OpenAI
- Binance API
- Bybit API
- и другие зависимости

## Установка
1. Склонируйте репозиторий
2. Установите зависимости
3. Запустите сервер
